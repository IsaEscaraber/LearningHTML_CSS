--------> EXPLICAÇÕES DE CADA PARTE DO HTML <-----------

  <!DOCTYPE html>   (serve para dizer o tipo do html).
  <html lang="pt-br'>  (mostra o idioma que o site será desenvolvido).
    <head>   (cabeça do site, onde fica as configurações do html).
      <meta charset="UTF-8"> (diz que o site precisará de acentos ou caracteres especiais do idioma).
      <meta name="viewport" 
            conntent="width=device-width, initial-scale=1.0">  (essa parte serve para configurar o site para abrir no 100%, sem zoom, e preenchendo a tela toda).
      <title> ANOTAÇÕES PARA LEMBRAR DEPOIS </title> (Texto apresentado na aba do site, no navegador). 
    <head>
    <body>   (corpo do site, onde fica o conteudo).
      <h1> OLÁ MUNDO!</h1>
    <body>
  <html>

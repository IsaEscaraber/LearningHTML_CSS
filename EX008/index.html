Main formats

Bold/ Contrast

In this sentese we wil have Bold term using the tag <b> (no semantic, without meaning).

Already in this other, I have a Contrast term using a <strong> tag (semantic, with meaning and reason).

Italic/ Emphasis

In this sentence, we have a italic term using the <i> tag (it's not semantic).

But in this other, we have emphasis term using a <em> tag (semantic, because EM means emphasis).

Marked Text

We can also create a highlighter term using a <mark> tag.

And in the other paragraph, we have another mark text in the end.

Big and small text

We are creating a texto grande using the <big> tag (that is already obsolete, outdated...) and a smal text using the <smal> tag (This is instead used).

Deleted Text

We can mark a text as excluded to indicate that it must be read, but does not consideted, using the <del> tag.

Inserted Text

We also can mark a text as inserted to give a emphasize and indicate that it was added before, using the <ins> tag (sematic way).

There is the underlined text too, with the <u> tag (no semantic).

Superscribed Text

It is to insert something of type x20+3, using the <sup> tag.

Subscribed Text

It is to insert something of type H2O, using the <sub> tag.
